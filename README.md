# React+PostCSS+CircleCIのテンプレートです
- create-react-appで作成しています．
- CircleCIはビルドしてテストを行い，Herokuへとデプロイする設定になっています．
- PostCSSはNestedとimportを入れています．

How to install

```
$ npm install -g create-react-app
$ npm install
or
$ yarn install
```

How to build

```
$ npm run build
or
$ yarn build
```

How to start

```
$ npm run start
or
$ yarn start
```
